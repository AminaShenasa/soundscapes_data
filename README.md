# Soundscapes Data

This is the GitHub repository for a project led by Amina Shenasa. The purpose of this project is to integrate ecological theory, soundscape ecology, and machine learning to address a key gap in ecosystem monitoring: the detection of dynamic but critical ecological transitions.

In this study, we tested whether soundscapes can serve as reliable indicators of kelp forest ecosystem conditions in Sitka Sound, Alaska. Specifically, we addressed whether (1) acoustics can distinguish between kelp forest and urchin barren habitats, (2) whether this distinction can detect varying levels of habitat type, and (3) whether the sounds discriminating habitat types are made up at least in part by urchin grazing acoustics. To address these questions, we collected data in Sitka, Alaska, conducting paired acoustic and ecological surveys at giant kelp forests, Macrocystis pyrifera, bull kelp forests, Nereocystis luetkeana, and urchin barren habitats. Additionally, we recorded urchin grazing acoustics to understand their role in the broader soundscape. 

By evaluating how traditional ecological indicators can be predicted by acoustic metrics in assessing kelp forest health, and how these ecological variables interrelate and manifest acoustically, this study explores how integrating soundscape data with traditional monitoring may enhance our ability to detect and respond to ecological change, ultimately improving ecosystem management and conservation efforts.
